# Mock Interview Script — Bilingual Panel / 中英双语交叉面试

## Purpose

- 训练候选人在中文解释技术细节、英文回答追问时的切换能力。

## Panel Format

- Round 1（中文）: 流程、floorplan、STA
- Round 2（English）: timing closure, debugging, trade-offs
- Round 3（双语混合）: project deep dive and behavioral

## Example Questions

- 中文：请解释 false path 与 multicycle path 的使用前提。  
  English follow-up: How do you validate that the exception is functionally justified?
- 中文：如果一个区域出现严重 IR drop，你会怎么排查？  
  English follow-up: Which fixes are fast but risky, and which are slower but robust?
- 中文：说一个你处理过的最难的 ECO。  
  English follow-up: What was the rollback plan and how did you de-risk the change?
- 中文：你如何与 top integration 团队合作解决跨 block timing？  
  English follow-up: What evidence would you bring to align different owners?

## Scoring

- Technical accuracy
- Structured communication
- Concise bilingual switching
- Ownership and prioritization
