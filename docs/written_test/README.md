# Written Test Zone Guide

## Purpose

- 这一目录专门服务于 `公开可追溯笔试题 + NVIDIA 风格 OA/笔试模拟卷` 的浏览与训练。

## Files

- `data/written_test/source_index.csv`：公开来源总表，标记是 `public-traceable` 还是 `nvidia-style`
- `data/written_test/public_traceable_written_sources.md`：逐条解释为什么收录这些公开来源
- `data/written_test/nvidia_style_written_bank_216.md`：216 道 NVIDIA 风格笔试题，含中英对照与答案要点
- `data/written_test/mock_exam_set_a.md`：90 分钟模拟卷 A
- `data/written_test/mock_exam_set_a_answers.md`：模拟卷 A 答案要点
- `data/written_test/mock_exam_set_b.md`：100 分钟模拟卷 B
- `data/written_test/mock_exam_set_b_answers.md`：模拟卷 B 答案要点

## Suggested order

1. 先看 `source_index.csv` 建立来源可信度概念
2. 再读 `public_traceable_written_sources.md` 了解哪些是公开线索，哪些是风格化扩展
3. 然后按主题刷 `nvidia_style_written_bank_216.md`
4. 最后做 `mock_exam_set_a.md` 和 `mock_exam_set_b.md`
